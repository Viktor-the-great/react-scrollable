Use css variables to customize track or/and thumb:

```css
.scrollable {
  /* thumb variables */
  --thumb-border: none;
  --thumb-border-radius: 3px;
  --thumb-background: #C7CED480;
  --thumb-size: 6px;

  /* track variables */
  --track-background: none;
  --track-border: none;
  --track-border-radius: 0;
}
```
